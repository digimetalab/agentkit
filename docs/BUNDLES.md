# 📦 AgentKit Meta-Bundles

Don't know where to start? Pick a Meta-Bundle to get a curated set of skills for your role.

## Installation

```bash
npx @digimetalab/agentkit
# Select your agent, then pick a bundle
```

Or install directly:
```bash
agentkit install foundation
```
